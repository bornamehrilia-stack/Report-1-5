### ۱. هدف آزمایش

آشنایی با خواندن مقادیر دیجیتال از ورودی، کار با ساختارهای شرطی در برنامه‌نویسی میکروکنترلر و درک مفهوم وضعیت‌های منطقی High و Low در مدارات آردوینو.

### ۲. شرح مختصر مدار

در این پروژه، یک مدار ورودی-خروجی طراحی شده است. ورودی مدار یک کلید فشاری (Push Button) است که به پین **۸** متصل شده و خروجی یک LED است که از پین **۲** فرمان می‌گیرد. مدار کلید به صورت فعال-پایین (Active Low) طراحی شده است؛ یعنی در حالت عادی ولتاژ پین ورودی High است و با فشردن کلید، مقدار `LOW` توسط آردوینو خوانده شده و دستور روشن شدن LED صادر می‌گردد.

### ۳. موارد مورد نیاز

* یک عدد برد **Arduino Uno**
* یک عدد **LED** 
* یک عدد کلید فشاری (**Push Button**)
* دو عدد مقاومت (یک عدد برای حفاظت LED و یک عدد به عنوان Pull-up برای کلید)
* برد بورد و سیم‌های پرشی (Jumper Wires)

### ۴. نمایش کد پروژه

```cpp
int buttonPin = 8;    // تعریف پین ورودی کلید
int ledPin = 2;       // تعریف پین خروجی LED
int buttonState = 0;  // متغیر ذخیره وضعیت لحظه‌ای کلید

void setup() {
  pinMode(ledPin, OUTPUT);    // پیکربندی پین 2 به عنوان خروجی
  pinMode(buttonPin, INPUT);  // پیکربندی پین 8 به عنوان ورودی
}

void loop() {
  // خواندن وضعیت منطقی پین 8
  buttonState = digitalRead(buttonPin);

  // منطق برنامه: اگر کلید فشرده شود (وضعیت LOW)
  if (buttonState == LOW) {
    digitalWrite(ledPin, HIGH); // روشن کردن LED
  }
  else {
    digitalWrite(ledPin, LOW);  // خاموش کردن LED
  }
}

```

### ۵. توضیح مختصر کد
.
* **تابع `digitalRead`:** این دستور به طور مداوم وضعیت ولتاژ پین ۸ را چک کرده و آن را در متغیر `buttonState` ذخیره می‌کند.
* **منطق شرطی:** از آنجایی که مدار به صورت Pull-up بسته شده، فشردن دکمه باعث اتصال پین به زمین (GND) می‌شود. بنابراین شرط `if(buttonState == LOW)` به درستی فشردن دکمه را تشخیص داده و LED را روشن می‌کند.
